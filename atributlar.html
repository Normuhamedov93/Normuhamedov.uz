Barcha HTML atributlar jamlanmasi

Bu yerda siz HTMLning barcha asosiy atributlari va ularning vazifalari haqida ma'lumot topishingiz mumkin.

Hech narsa topilmadi.

1. Global Atributlar (Barcha teglarga xos)

  • class - Element uchun bir yoki bir nechta sinf nomlarini belgilaydi (CSS orqali uslub berish uchun).
  • id - Element uchun unikal (takrorlanmas) identifikator belgilaydi.
  • style - Element uchun inline (ichki) CSS uslublarini belgilaydi.
  • title - Element ustiga sichqoncha olib borilganda qo'shimcha ma'lumot ko'rsatadi.
  • lang - Element tarkibidagi matn tilini belgilaydi (masalan, "en", "uz").
  • hidden - Elementni brauzerda ko'rinmaydigan qilib qo'yadi.
  • tabindex - Elementning klaviatura orqali fokuslanish tartibini belgilaydi.
  • data-* - Sahifaga maxsus, shaxsiy ma'lumotlarni saqlash uchun ishlatiladi.

2. Havola va Media Atributlari

  • href (<a>, <link> uchun) - Havola manzilini (URL) belgilaydi.
  • src (<img>, <script>, <iframe> uchun) - Manba faylning yo'lini (URL) belgilaydi.
  • alt (<img> uchun) - Rasm yuklanmaganda yoki ko'zi ojiz foydalanuvchilar uchun uning o'rniga ko'rsatiladigan alternativ matn.
  • width, height (<img>, <video> uchun) - Elementning kengligi va balandligini belgilaydi.
  • target (<a> uchun) - Havola qayerda ochilishini belgilaydi (masalan, _blank yangi oynada ochadi).
  • controls (<audio>, <video> uchun) - Media uchun boshqaruv panellarini (play, pause, ovoz) ko'rsatadi.
  • autoplay (<audio>, <video> uchun) - Media yuklanishi bilan avtomatik ravishda ijro etilishini ta'minlaydi.

3. Forma Atributlari

  • action (<form> uchun) - Forma ma'lumotlari yuboriladigan manzilni (URL) belgilaydi.
  • method (<form> uchun) - Ma'lumotlarni yuborish usulini belgilaydi (GET yoki POST).
  • type (<input> uchun) - Kiritish maydonining turini belgilaydi (masalan, "text", "password", "submit").
  • name - Element nomini belgilaydi (serverga ma'lumot yuborish uchun kerak).
  • value - Kiritish maydonining dastlabki qiymatini belgilaydi.
  • placeholder - Kiritish maydonida ko'rsatiladigan maslahat matni.
  • required - Maydonni to'ldirish majburiy ekanligini bildiradi.
  • disabled - Elementni o'chirib qo'yadi, foydalanib bo'lmaydi.
  • readonly - Maydon qiymatini faqat o'qish uchun qoldiradi (o'zgartirib bo'lmaydi).
  • maxlength - Kiritish maydoniga kiritilishi mumkin bo'lgan belgilarning maksimal sonini belgilaydi.
