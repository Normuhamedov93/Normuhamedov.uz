JavaScript bo'yicha qo'llanma

Bu yerda siz JavaScriptning eng ko'p qo'llaniladigan tushunchalari haqida ma'lumot topasiz.

Hech narsa topilmadi.

1. O'zgaruvchilar va Ma'lumot Turlari

  • let - O'zgaruvchan qiymatga ega bo'lgan blok-sohasidagi o'zgaruvchi e'lon qilish.
  • const - O'zgarmas qiymatga ega bo'lgan (konstanta) o'zgaruvchi e'lon qilish.
  • Ma'lumot turlari: String (matn), Number (son), Boolean (true/false), Object (obyekt), Array (massiv), null, undefined.

2. Operatorlar

  • Arifmetik: +, -, *, /, % (qoldiq), ** (daraja).
  • Taqqoslash: == (teng), === (qat'iy teng - qiymat va tur bo'yicha), !=, !==, >, <.
  • Mantiqiy: && (VA), || (YOKI), ! (INKOR).

3. Shartli Operatorlar va Sikllar

  • if...else - Shartga qarab kod blokini bajarish.
  • switch - Ko'p holatlardan birini tanlab bajarish.
  • for - Belgilangan marta takrorlanuvchi sikl.
  • while - Shart bajarilguncha takrorlanuvchi sikl.
  • .forEach() - Massivning har bir elementi uchun funksiyani ishga tushiradi.

4. Funksiyalar

  • Function Declaration: function nom(parametr) { ... }
  • Arrow Function (Strelkali funksiya): const nom = (parametr) => { ... } - Zamonaviy va qisqa yozuv.

5. DOM (Document Object Model) bilan ishlash

  • document.getElementById('id') - Elementni ID orqali topish.
  • document.querySelector('.class') - Elementni CSS selektori orqali topish (birinchisini).
  • document.querySelectorAll('.class') - Barcha mos elementlarni topish.
  • .addEventListener('click', funksiya) - Elementga hodisa (masalan, bosish) qo'shish.
  • .innerHTML, .textContent - Elementning ichidagi kontentni o'zgartirish.
  • .style.property - Elementning CSS uslubini o'zgartirish (masalan, .style.color = 'red').
  • .classList.add('new-class'), .classList.remove('class') - CSS sinflarini qo'shish va o'chirish.

6. Asinxron JavaScript

  • setTimeout(funksiya, vaqt) - Belgilangan vaqtdan so'ng funksiyani bir marta bajaradi.
  • Promise - Asinxron operatsiyaning yakuniy natijasini ifodalovchi obyekt. .then() va .catch() bilan ishlanadi.
  • async/await - Promise'lar bilan ishlashni osonlashtiruvchi sintaksis. Kodni sinxron ko'rinishda yozishga imkon beradi.
